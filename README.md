# Praveen Kumar - AI Engineer Portfolio

A stunning, professional portfolio website built with **Vite + React** featuring a timeless design with navy blue, royal black, soft white, and gold accents. Perfect for showcasing AI/ML engineering expertise.

## 🎨 Design Features

- **Color Palette**: Navy Blue, Royal Black, Soft White, Gold Accents
- **Typography**: Playfair Display (headers) + Inter (body)
- **Style**: Classic, professional, timeless with futuristic AI elements
- **Responsive**: Optimized for desktop, tablet, and mobile
- **Animations**: Smooth transitions using Framer Motion
- **UI Elements**: Neural network patterns, geometric lines, floating particles

## 🚀 Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation & Running

1. **Install Dependencies**
```bash
npm install
```

2. **Start Development Server**
```bash
npm run dev
```

3. **Build for Production**
```bash
npm run build
```

4. **Preview Production Build**
```bash
npm run preview
```

## 🎨 Customization

### Update Personal Information

Edit the content in each component file:
- `Hero.jsx` - Name and introduction
- `About.jsx` - Personal description
- `Experience.jsx` - Work experience
- `Skills.jsx` - Technical skills
- `Projects.jsx` - Portfolio projects
- `Education.jsx` - Education and certifications
- `Contact.jsx` - Contact information

### Change Colors

Edit CSS variables in `src/index.css`:
```css
:root {
  --navy-blue: #0A1929;
  --royal-black: #1A1A2E;
  --soft-white: #F8F9FA;
  --gold-accent: #D4AF37;
  --light-gold: #E8D7A0;
}
```
